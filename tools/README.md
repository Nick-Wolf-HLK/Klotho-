# tools/

Build-Werkzeug für die ASCII-Art-Assets von Klotho.

## `build_art.py`

Erzeugt `klotho/art.py` (Logo + Göttinnen-Porträt) als statische String-Konstanten.

```bash
pip install pillow        # nur zum Bauen nötig, keine Laufzeit-Abhängigkeit
python tools/build_art.py
```
